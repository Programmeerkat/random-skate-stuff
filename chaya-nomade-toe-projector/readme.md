# Chaya Nomade toe protector

## Description
The toe is arguably the weakest part of the Chaya skates. This toe projector can be slit on and attached using a few pieces of double-adhesive tape. 

## Printing instructions
I've tested these printed with ABS with using a 0.4 mm nozzle, with a 5 bottom layers, 5 top layers, 5 walls, and 50% infill.
